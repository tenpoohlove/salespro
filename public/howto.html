使い方ガイド
Pitch Navi の使い方・分析できる素材・困ったときの対処をまとめています。
かんたん3ステップ
APIキーを設定
右上の「 キー設定」から、分析用の Anthropic キー（必須）と、文字起こし用の OpenAI キー（動画・音声を使う場合）を入力します。各テストボタンで有効性を確認できます。
クロージング商談を入れる
Zoom商談の録画（.mp4）・録音（.mp3/.m4a）をアップロード、または会話の文字起こしをテキスト欄に貼り付けます。動画・音声は自動で文字起こしされます。
クロージング添削する
「 クロージングを採点・添削する」を押すと、スコアと改善提案（言い回しの具体例つき）が出ます。必要なら「商談の備考」に相手の情報を補足できます。
分析できる素材
セールス動画（.mp4 / .mov）
アップロードすると自動で文字起こし → そのまま分析。25MB以下推奨。
セールス音声（.mp3 / .m4a / .wav）
アップロードすると自動で文字起こし → そのまま分析。
スライド資料
.pptxファイルを直接アップロード可能。スライド画像（.png/.jpg）も対応
セールスページ / LP
コピーテキストを貼り付け、またはPDF/DOCXをアップロード
セールスメール
メール本文をテキストエリアに貼り付け
困ったときは
動画が大きすぎてアップロードできない（25MB超）
動画から音声だけ書き出してください。
Windows: VLCを開く → メディア → 変換/保存 → ファイル追加 → 変換/保存 → プロファイルで「Audio - MP3」を選択 → 保存
Mac: QuickTimeで動画を開く → ファイル → 書き出す → オーディオのみ
文字起こしがすでにある場合
動画・音声をアップロードしなくてもOKです。テキストエリアに文字起こしを貼り付けるか、.txtや.srtファイルをそのままアップロードしてください。
文字起こしに時間がかかる / エラーになる
ファイルが大きいほど時間がかかります。エラーになった場合はファイルサイズを確認してください（上限25MB）。
URLから読み込みたい
「URLから取得」にWebページのURLを入力するとテキストを自動取得して分析できます。
APIキーはどこで取得する？
分析用：console.anthropic.com（Anthropic）
文字起こし用：platform.openai.com（OpenAI）
