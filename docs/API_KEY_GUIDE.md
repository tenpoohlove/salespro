# APIキー取得ガイド（ご利用者向け）

このツールは、**AIの利用料がご自身のAPIキーに直接かかる仕組み（BYOK）** です。
運営側に費用が乗らない代わりに、最初に一度だけ、お使いのAPIキーをご用意いただきます。
入力したキーはご自身のブラウザ／アカウントのために使われ、分析以外には使いません。

必要なキーは最大3種類。**まずは ① だけあれば、商談の添削（テキスト分析）が使えます。**

| キー | 何に使う | 必須？ |
|------|----------|--------|
| ① Anthropic（Claude） | 商談の分析・添削 | **必須** |
| ② OpenAI（Whisper） | 音声・動画を自動で文字起こし | 文字起こしを使うなら必要 |
| ③ Fish Audio | 「本人の声」での理想クロージング音声生成 | 声クローンを使うなら必要 |

---

## ① Anthropic（Claude）APIキー … 分析に必須

1. https://console.anthropic.com/ にアクセスし、アカウント作成／ログイン
2. 左メニューの **「API Keys」** を開く
3. **「Create Key」** をクリックし、名前を付けて作成
4. 表示された **`sk-ant-...` で始まる文字列をコピー**（※この画面でしか表示されません。閉じる前に必ずコピー）
5. 事前に支払い設定が必要です: **「Billing」→「Plans」/「Buy credits」** でクレジットを購入（最初は$5〜でOK）

**料金の目安**: 商談1本の添削で **およそ10〜20円**。同じ内容の2回目はキャッシュで無料。

---

## ② OpenAI APIキー … 音声・動画の自動文字起こしに使用

※ すでに文字起こし済みのテキストを貼り付けて使う場合は不要です。

1. https://platform.openai.com/ にログイン
2. 右上のメニュー → **「API keys」**（または https://platform.openai.com/api-keys ）
3. **「Create new secret key」** → 名前を付けて作成
4. **`sk-...` で始まるキーをコピー**（再表示されないので必ず保存）
5. **「Billing」** で支払い方法を登録し、少額チャージ（最初は$5〜）

**料金の目安**: 文字起こしは **1分あたり約1円**。40分の商談で **約40円** ほど。

---

## ③ Fish Audio APIキー … 声クローン（任意）

※「本人の声で理想クロージングを聞く」機能を使う場合のみ。

1. https://fish.audio/ にアクセスしてログイン
2. アカウント設定 → **「API Keys」** からキーを発行してコピー
3. 課金設定（クレジット購入）を済ませる

**料金の目安**: 1回の音声生成で数円程度。同じ台本の2回目はキャッシュで無料。

---

## ツールへの入れ方
1. ツールにログイン後、画面上部の **APIキー入力欄** にコピーしたキーを貼り付け
2. キーはお使いのアカウントに紐づいて保存され、毎回入れ直す必要はありません

## よくある質問
- **Q. キーを盗まれたら？** → 各サービスの管理画面でそのキーを「削除（Revoke）」すれば即無効化できます。新しいキーを作り直してください。
- **Q. なぜ運営のキーで動かさないの？** → 利用が増えても運営に費用が集中せず、各自の使った分だけを各自が負担できる公平な仕組みにするためです。
- **Q. キーの料金が高くならない？** → 高額モデルを必要な場面だけに絞り、同じ入力の再分析はキャッシュで無料にしています。通常利用なら1日数十円〜数百円の範囲です。
- **Q. 最低限どれが要る？** → ① Anthropic だけで、テキストを貼り付けての商談添削はすぐ使えます。
