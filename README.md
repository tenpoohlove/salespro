# セールスアドバイザー｜成約率アップ AI分析ツール

セールス動画・スライド・LP・メールなどを分析し、成約率を上げる具体的なアドバイスを生成するツールです。

---

## はじめにやること（初回セットアップ）

### 1. Node.js のインストール

Node.js がインストールされていない場合は、下記からインストールしてください。

**https://nodejs.org/ja/**（LTS版を選択）

インストール後、コマンドプロンプトで確認：
```
node --version
```
`v18.x.x` 以上が表示されれば OK です。

---

### 2. Anthropic API キーの取得

このツールは Anthropic（Claude）の AI を使用しています。  
**利用料金はあなたのアカウントに課金されます。**

1. **https://console.anthropic.com/** にアクセス
2. アカウントを作成（または既存アカウントでログイン）
3. 左メニュー「API Keys」→「Create Key」
4. 生成されたキー（`sk-ant-api03-...`）をコピーして保管

---

### 3. ツールの起動

フォルダ内の **`start.bat`** をダブルクリックしてください。

初回起動時は自動的に必要なパッケージをインストールします（数分かかります）。

黒い画面に以下が表示されたら起動完了です：
```
🚀 セールスアドバイザー起動中
   http://localhost:3000
```

---

### 4. API キーの設定

ブラウザで **http://localhost:3000** を開くと、APIキー入力欄が表示されます。

取得した APIキー（`sk-ant-api03-...`）を入力して「保存」を押してください。

**一度保存すれば次回以降は不要です。**

---

## 使い方

1. ブラウザで http://localhost:3000 にアクセス
2. 分析したい素材をアップロード（または本文をテキストエリアに貼り付け）
3. 分析フォーカスを選択（総合 / Hook強化 / CTA重点 / 信頼構築）
4. 「成約率を分析する」をクリック

---

## 対応ファイル形式

| 形式 | 用途 |
|------|------|
| `.pdf` | セールスページ、提案書 |
| `.pptx` | プレゼン資料、スライド |
| `.docx` | セールスレター、原稿 |
| `.txt` / `.md` | テキスト原稿 |
| `.srt` / `.vtt` | 動画の字幕ファイル（文字起こし） |
| `.png` / `.jpg` / `.webp` | スライド画像、LPスクリーンショット |

動画・音声ファイルは直接アップロードできません。  
文字起こしをテキストエリアに貼り付けてご使用ください。

---

## 分析内容

- **総合スコア**（100点満点）
- **10要素 詳細スコアリング**（Hook / Problem / Solution / Authority / Social Proof / Benefits / CTA / Urgency / Objections）
- **TOP 3 即効改善ポイント**（インパクト順、実装コスト付き）
- **改善スクリプト例**（そのまま使えるコピー）
- **心理的フック強化案**（日本市場特化）
- **A/Bテスト推奨案**
- **実装チェックリスト**

## よくある質問

**Q. ツールを閉じたらデータは消えますか？**  
A. 分析履歴はブラウザに保存されます（最新10件）。黒い画面（サーバー）を閉じても履歴は残ります。

**Q. 料金はどれくらいかかりますか？**  
A. 1回の分析で約 $0.01〜$0.05（数円〜数十円）が目安です。Anthropic のダッシュボードで使用量を確認できます。

**Q. ファイルはどこかに送信されますか？**  
A. ファイルはあなたのパソコン内で処理され、Anthropic の API のみに送信されます。第三者には送信されません。

**Q. APIキーを変更したい場合は？**  
A. http://localhost:3000 を開いた際、もしくはフォルダ内の `.env` ファイルをメモ帳で開いて直接書き換えてください。
